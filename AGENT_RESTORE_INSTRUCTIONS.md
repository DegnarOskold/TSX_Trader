## 4. Operational Notes
* **Powershell Quirk:** When using the Agent to execute `send_telegram.py`, double quotes (`"..."`) cause Powershell to evaluate `$` signs as variables (e.g. `$63.40` becomes `.40`). It is strictly required that the agent wraps the telegram string in single quotes (`'...'`).
* **Ledger Updates:** Do not manually edit `trades.csv` unless explicitly asked. The daemon handles it automatically.
* **Analysis Formatting:** Do not include RSI or SMA in the recommendations. The user specifically prefers to only see Average Purchase Price vs Current Price.
